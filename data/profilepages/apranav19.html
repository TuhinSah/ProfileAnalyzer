  • Apache Web Server Log Analysis

    Starting June 2015

    This was my first "Big Data"​ project where I utilized PySpark, a Python API for Apache Spark, to analyze and visualize data from production logs for NASA Kennedy Space Center's web server.

    The various analyses that I performed include:

    * Determining the most frequent hosts that have accessed the server on multiple occasions
    * Visualizing the number of hits to the various URIs in the logs
    * Determining the number of unique hosts on a daily basis
    * Computing the average number of HTTP requests per unique host
    * Discovering the top 25 hosts that generated the most HTTP 404 errors

    Team members:
    • Pranav Angara

  • AU Metriks

    Starting September 2015

    AU Metriks - A web based dashboard that provides insights into Autodesk University's (http://au.autodesk.com/) class content & attendee data.

    Using Autodesk University's class content data from 2012 - 2014 (~3500 classes) and the conference attendee data from 2014 (~9000 attendees), we extracted a number of insights such as:
    - Segmentation of classes offered by industry type
    - Segmentation of classes offered by Autodesk product/software
    - Segmentation of conference attendees by geography
    - Segmentation of conference attendees by profession

  • Styra

    Starting September 2014

    Styra, Swedish for "Steer"​, is a navigation service that provides driving/walking/cycling directions via text messaging.
