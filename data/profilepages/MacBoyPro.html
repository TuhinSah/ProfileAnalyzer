  • Co-Founder/Software Engineer

    Blazing Cloud
    June 2009 – November 2013 (4 years 6 months)

    As a software consultant I got to work with a range of clients, on software products that span web, native desktop, and mobile.

    LeapMotion, led the development of their payments infrastructure used by the LeapMotion AppStore. Payments was a micro-service written in Ruby on Rails, utilizing BrainTree as it's payment processor.

    Rabbit, led the development of a UIKit like framework written in OpenGL/Objective-C for their Mac OS X video streaming application. The framework was responsible for rendering, animating and I/O.

    Adobe CC, helped prototype their first Creative Cloud iOS application, which included the ability sync files from the user's CC to their device and interact with those files.

    FotoZap, an event marketing tool to capture branded photos and videos. Developed their first application to upload photos and videos that correspond to a bar code. The user would take a picture using the app, scan the barcode and hand it to the guest for later retrieval on the web.

    Sparky, a Mac OS X desktop application that aggregated the user's Facebook and Twitter feeds into one intuitive UI.

    Over my time at Blazing Cloud, I got to work on and develop over 10 iOS applications, including iPad, iPhone and universal applications, both connected and offline, featuring sound, video, web views and localization. Write both mobile and desktop Web applications with JavaScript/JQuery and Ruby on Rails 2.2-3.1. Contribute to OpenSource frameworks like Rhomobile Rhodes, cross platform mobile framework. Develop curriculum and training in HTML/CSS, JavaScript and Ruby on Rails. Work with many third party services including the Facebook and Twitter APIs.

  • Software Engineer Intern

Publications

  • Pro Smartphone Cross-Platform Development: iPhone, Blackberry, Windows Mobile and Android Development and Distribution

    APRESS
    September 2010

    Learn the theory behind cross-platform development, and put the theory into practice with code using the invaluable information presented in this book.
