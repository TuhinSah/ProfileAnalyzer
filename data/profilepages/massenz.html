I build great teams who build great products.

Leading from the front, with a deeply technical, hands-on knowledge of how to design and implement massively scalable distributed systems in both on-premise, virtualized and Cloud environments.

I also have a proven track record in turning around troubled engineering organizations, attracting talented engineers to break monolithic, untestable enterprise applications into API-driven, loosely coupled microservices, which can be deployed using fully automated CI/CD techniques, with great test coverage.

  • VP Engineering and Chief Architect

    I personally designed, developed and maintained the external (public) REST API, as well as worked closely with the team to design and implement the internal (private) REST API between the Java frontend and the Python backend (Migration Engine).
