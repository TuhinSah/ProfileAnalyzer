Summary

Notable Personal Achievements:

Originally learned to program by myself at age 9, without Internet access (or books) during my first year with a computer, by reverse engineering batch files.

At age 10, I began game programming in C++, which would start a lifelong career and hobby.

At Age 18, I developed a game aimbot using pixel scanning and edge detection. I also developed a speedhack using an input system exploit.

Started company in 2009, as sole developer of two shipped enterprise products. Worked with and provided both native and web software development for numerous businesses.

Open-source developer and contributer to numerous projects spanning many languages and technologies. (See GitHub profile)

Successfully cofounded and operated a flight training company for many years.

My company was discovered by a Google recruiter when I was 24.
