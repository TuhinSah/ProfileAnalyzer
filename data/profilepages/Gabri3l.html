  • Growing Hierarchical Self-Organizing Maps for Document Image Clustering

    Starting October 2015

    This thesis work presents two different approaches for historical document image clustering. The first one makes use of feature engineering to extract layout traits, using information on the input data domain and then grouping images with K-Medoids clustering algorithm. The second approach harnesses the power of machine learning methods employing Denoising Autoencoders and Convolutional Neural Networks to extract robust features, these new image representations are then adopted in a Growing Hierarchical Self-Organizing Map to acquire a topologically meaningful clustering. We also provide a graphical user interface to tackle the problem of work review, allowing the user to mold the intended clustering and attain the required quality level and cluster purity. In this project I used both C and Python.

    Team members:

  • Implementation of a large-scale image retrieval technique with compressed Fisher vectors

    Starting April 2015

    This implementation originates from a work of Perronnin et al. facing the problem of large-scale image search. They address the issue with an approach based on the Fisher kernel framework. Starting from an image represented as a set of local descriptors, the patch encoding strategy frequently used in other works is the popular Bag-of-Visual words (BOV) representation. Perronnin et al. propose a different patch encoding strategy that makes use of the Fisher kernel framework, proving to be well-suited for the retrieval problem: it describes an image focusing on what makes it different from other images, it is efficient to compute, and it can be compressed with a minimal loss of accuracy. In our case of study we are concerned with the problem of compression, which characterizes one of the main drawbacks that make Fisher vectors generally not directly amenable to large-scale retrieval. We compare their novel binarization technique devised for this representation and two standard compression techniques. Our project is implemented in Matlab with the support of Yael library. We show how we were able to achieve the same results showed as Peronnin et al. highlighting how the compressed Fisher vectors perform very well, especially when compared to the most common BOV approach with up to a 10% improvement in Mean Average Precision.

    Team members:
    • Gabriele Cimato,
    • Lorenzo Carli
