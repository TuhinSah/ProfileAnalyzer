  • CSS Casino

    Starting September 2016

    CSS Casino demos some of the cool interactive animations you can do with purely HTML and CSS.

    For this project, I laid down these constraints to test how far I can stretch usage of HTML/CSS in creative ways. These animations not only convey meaning and purpose through movement but also demonstrates how CSS transitions and animation properties work. CSS Casino was completed over the span of 4 mornings.

    Team members:

  • Activated Railroad

    Starting September 2016

    Activated Railroad is an ORM and web server MVC inspired by ActiveRecord and Rails respectively.

    Curious about the inner workings of ActiveRecord and Rails, I metaprogrammed frameworks inspired by each for this project. Activated is an ORM capable of translating rows from a SQL query into Ruby objects (and vice versa) and also handling model class associations. As a web server MVC, the Railroad framework supports functionalities such as template rendering, session handling, routing and CSRF protection.

    Team members:
    • Grace Patuwo
