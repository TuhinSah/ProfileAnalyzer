Experience

  • Software Engineering Intern

    Numenta
    May 2016 – August 2016 (4 months)

    Wrote open source software for Numenta's Platform for Intelligent Computing NuPIC. Some of the major efforts I worked on was redesigning the temporal memory in NuPIC, an algorithm used to learn temporal patterns in streaming data and a core pillar of Numenta's work with biologically plausible neural networks. Achieved a 30% speedup in the temporal memory and ensured 100% parity with the C++ implementation, allowing for a standard to be set for future implementations of the temporal memory in other languages. Also added new features and improved existing features in Numenta's Network API for creating custom HTM networks from the ground up. All of my code can be seen on Numenta's github: https://github.com/numenta primarily in the nupic and nupic.core repos.
