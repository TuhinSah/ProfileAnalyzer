Projects

  • Journey

    Journey is a web app to help you plan your career. It gives actionable insights on how to obtain a desired job by collecting, analyzing, and visualizing the career trajectory data of people similar to you on LinkedIn.

    As product owner of this application, I:
    - Designed and built the front-end architecture with Backbone.js
    - Created an attractive user interface with Bootstrap and CSS
    - Designed database structure and set up MySQL database using Knex and Bookshelf
    - Configured the client to communicate with a REST API and enabled retrieval of career data from the database
    - Implemented functionality for search and filtering with jQuery UI
    - Developed dynamically generated charts that respond to user input using HighCharts
    - Wrote serverside and client-side tests with Jasmine


    Team members:
    • Sunshine Yin,
    • Derek Boero,
    • Yuriy Bash

  • QWERTY Piano

    QWERTY Piano turns your computer keyboard into a mini-piano keyboard so you can practice typing while playing piano tunes. I designed and built the user interface with jQuery and Semantic UI and wrote the algorithm for matching QWERTY keys to music notes. I also developed an algorithm to sync the time signature of music pieces with the display of keystroke prompts.

    Team members:
    • Sunshine Yin,
    • Darren Wong

  • SocialCap (Social Capital)

    SocialCap is a stock market game to bet on changes in public sentiment toward trending topics.

    As a full-stack engineer on this team, I configured the Node/Express server to query the Twitter API for data that was stored in a MongoDB database. We used Indico.io to analyze the sentiment of tweets. We translated a large dataset into meaningful correlations between hashtags, tweet sentiment, and tweet reach in order to generate real-time stock price fluctuations. I also designed and implemented the visualization of this data analysis with Chart.js.

    Team members:
    • Sunshine Yin,
    • Cheng Tian,
    • Quest Henkart,
    • Darren Wong

  • Vouchery

    Vouchery is a web app that helps you find and share referral codes with Facebook friends.

    I helped build the UI with the React.js framework and Flux architecture, including features to add, share, view, and search for referral links. I also integrated the front-end with a Node/Express server and SQLite/Sequelize database.

    Team members:
    • Sunshine Yin,
    • Cheng Tian,
    • Darren Wong,
    • Quest Henkart,
    • George Lam

  • Should I Have Another Drink?

    This was a fun personal side project. I designed and built an app using AngularJS and Bootstrap on the frontend, and Node/Express on the backend. I wrote an algorithm for determining a user's level of inebriation based on their input to prompts.
