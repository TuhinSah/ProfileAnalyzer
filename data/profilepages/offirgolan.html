Projects

  • ResNet Bandwidth Monitor (PURU) - Available for Download

    Starting December 2012

    An elegant and minimalistic Mac OS X bandwidth monitoring application for the Purdue University dorm residents. Available for download by the Purdue ResNet Team.

    Features include:
    • Real time network monitoring
    • Automatic updates to retrieve the latest data regarding user's data usage from the ResNet internet provider
    • User defined warning and critical alerts to notify them about certain bandwidth limits
    • Refresh algorithm to reduce traffic on the ResNet data website
    • Lightweight application that sits in the menu bar. Takes 0% CPU and does not reduce battery life
    • Senses when user has disconnected and reconnected to the network to save battery and network traffic

    Original Design can be found here: https://github.com/offirgolan/Bandwidth-Monitor-Unofficial
