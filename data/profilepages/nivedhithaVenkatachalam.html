  • Predictive Analysis in Movie Industry

    January 2015 – May 2015

    The project ‘Predict Movie Ratings’ aims to provide a better understanding of the trends in movie industry by using social media analysis, to predict individual user’s rating for movies and provide personalized suggestions to each user. The end product of the project is a website where the user has to register first. Once he registers and signs in, he gets randomized suggestions. The user can also search for movies by using a keyword search. The details of the movies are obtained by querying the IMDB API and the user can start rating them. Once the user has rated 20 movies, personalized suggestions are given to the user. Not only does our website predicts an individual user’s liking but also it displays the statistics on how the movie is currently trending. This is done by collecting information from social media sites. Examples include collecting the number of likes from Facebook, views from YouTube, ratings provided to the movie in websites such as IMDB and rotten tomatoes, tweets from twitter and analyzing them for sentiment score. All these information are put together in graphs and charts for better understanding. By providing such details to the user, the user gets a better picture about the movie in a single portal.
    In order to develop the model for prediction and recommendation, we have used Singular value decomposition (SVD) along with cosine similarity technique. This model is created by using the movielens data set which consists of ratings given by users (user id), the movies rated by the users (movie ids) and the ratings given by these users. Using the past movies rated and the ratings to the movies by the users, SVD builds a matrix on which we can compute the cosine similarities to calculate the ratings for the other movies (unrated movies).

    • srinidhi srinivasa raghavan,
    • Akilesh Mani

  • Frogger Ninja – Multiplayer Online Game

    September 2015 – December 2015

    FroggerNinja is a multi player online game. The game is an advanced version of classic games of 80s “Frogger”, which was a single player game with objective to make the frog be able to cross the street full of heavy traffic. The project extends the game into a multiplayer version. The game is based is developed in ThreeJS framework. The technology behind it is WebGL, which is an extension to OpenGL 2.0. Sending game data to respective user from the server is handled using SocketIO. To display statistics of game results, open source libraries such as Google HighCharts and D3 JS is used.
