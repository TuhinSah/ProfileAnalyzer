  • Wedding R.S.V.P.

    April 2014 – March 2015

    Throughout the course of a year following the engagement to my now wife I had developed a website which would handle all guest communications. The site UI in itself was simple and very informational. Every guest received a physical invitation to the event with a personal username and password which would grant them access to the site. Once logged in they would be able to find out all event details as well as input the number of guests and meal options they desired.

    The technologies used for this project were HTML, CSS, JavaScript, PHP, and MySQL. I developed all frameworks and libraries used for this particular project from scratch. Images used were allocated through Creative Commons aggragates and were modified using GIMP as necessary.

    Currently the site is no longer active but the source code is available for viewing and use:
    https://github.com/scotmatson/wedding

    This project was an excellent study in web development and time management for myself. During the course of this project I was a full time student, employed part-time and planning a mid-sized wedding. There are many features that were unfortunately not implemented, however, the deployed infrastructure did serve its purpose and made our lives much easier while coordinating with our guests.
