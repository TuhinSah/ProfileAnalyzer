Projects

  • Lutris

    Starting October 2009

    Lutris is a fully open source gaming platform for Linux. It allows to mange and run any video game for almost every video game platform ever created. The client can install, configure and run games on any Linux platform while the website provides install scripts written by the community.
    The project distributes tools necessary to run games as well as free and open source games. We currently do not provide commercial games but the platforms integrates with other game stores such as Steam, GOG and Humble Bundle.
