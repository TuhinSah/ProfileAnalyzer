Summary

the science of data--collecting it, storing it, transforming it, displaying it, and using it for prediction.

designing, and implementing the infrastructure required to collect, clean, and store that data.

using analytical techniques to solve a variety of business problems--e.g., hyper-personalized user experience, predicting lifetime customer value (LTV), predicting online fraud, decomposing the population of registered customers into discrete, targetable segments.

fluent in writing production code for the services layer, computational and traditional statistical methods as well as Machine Learning techniques: (i) spectral decomposition techniques (e.g., kPCA); (ii) supervised techniques, e.g., support vector machine,multi-layer perceptron, decision tree(CART/C4.5) and FP-Growth/FP-Tree.

Many of these techniques, i have designed, built, and implemented to scale e.g., for real-time user recommendation, most recently using cloud resources AWS (SQS, S3, and EMR).

specialties: (i) C, and Python (+ computation libraries NumPy, SciPy, & MatPlotlib) JavaScript, & R; (ii) Fluent in DataWarehouse design (dimensional modeling), implementation, and operation; (iii) Statistics-based and Computational Geometry Machine Learning techniques; (iv) Combinatorial Optimization w/ meta-heuristics; (v) Numerical Methods (e.g., matrix decomposition); ((vii) hand-coding interactive Data Visualization web apps using Flask (python micro-framework), HTML5, CSS3, and Javascript.
