Projects

  • News Juice

    News Juice aggregates news and Google Trends API’s based on user search terms, and then renders data visualizations based on the results. This allows end users to see the prevalence, beginnings, and momentum of the keywords they searched for. This app was built using AngularJS, Node.js, Express, MongoDB, and the D3.js data visualization library.

    - As Scrum Product Owner for this application’s development, I drove the team’s vision for high-level product direction and functionality.
    - Built Node.js Express server from the ground up.
    - Refactored aggregate data visualization code to allow intelligent scaling of D3 graph based on real-time user response.
    - To scale the data visualizations, rather than create aspect ratio errors by using responsive CSS, my optimization configured AngularJS to clear out and re-render the graph to the SVG element dynamically, based on event listeners placed in raw HTML.

    Team members:

  • Memowise

    Memowise is an intelligent flash-card management application which uses spaced-repetition to help people learn efficiently. Memowise was built using React, Redux, Express.js, and MongoDB. To develop this app, I:

    - Created a site-wide search feature which allows users to narrow down the collection of study cards displayed on the page in real-time as they type.
    - To do this, implemented two-way data-binding in React / Redux and used the search input to filter out which collections would be dynamically rendered based on user input.

    Github: https://github.com/nligon/Memowise

    Team members:
    • Nathanael Ligon,
    • Ben Rachbach,

    • Tegan Duong

  • Farm Connect

    Farm Connect is a marketplace application for local farmers to advertise directly to users, and for users to have direct access to fresh produce at supplier prices. Farm Connect was built using Angular.js, Express.js, JWT, and PostgreSQL.

    - I was ScrumMaster for this application's development.
    - I safeguarded workflow process, style guide implementation, and documentation.

  • EVA

    EVA is an artificial intelligence that will chat with you from across the internet. I created her using an AI constructor, and made her accessible with an AngularJS interface. To develop this app, I:

    - Utilized api.ai to instantiate and configure the AI.

    - Handled authentication utilizing Express routing on a Node.js server.
