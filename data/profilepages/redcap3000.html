Summary

I am an open source developer with a background in graphic design / video production / post production, multimedia authoring, and Journalism and Public communication. I am also highly interested in the quantitative health movement.

My current work centers around the use of Javascript / node frameworks (specifically one called Meteor) for creating real-time single page web applications. Some API's that I have worked extensively include Google Maps, Instagram, Moves, and Strava.

Much of my open source work involves API development and the task of creating open-source medical documentation and billing systems.

Specialties: mongo, api/api library development, PHP 5, unix system administration/optimization, apache couch, SEO, web design,SQL, Javascript, Wordpress optimization,Graphic design, video production
