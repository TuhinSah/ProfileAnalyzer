  • Reliability Engineer

    Thermo King
    February 2012 – July 2013 (1 year 6 months)Greater Minneapolis-St. Paul Area

    Drove product reliability and provided reliability metrics to upper management.
    • Used an object oriented paradigm to build efficient and robust MATLAB tools designed to streamline and optimize the reliability program’s data acquisition and analysis methodologies. Developed Software to:
    - Automate the construction of SQL queries and download warranty data from an Oracle Data Warehouse.
    - Generate automated routine warranty reports.
    - Filter warranty data into failure and censor subsets based on part and subsystem.
    - Preform massive rapid Weibull analysis on all parts for all subsystems over manufacturing eras and thus to forecast warranty risk to the business.
    - Extrapolate the minimum cut-sets from a system architecture as described by a Binary Decision Diagram (BDD) and simulate the limiting system availability through a two state Markov chain Monte Carlo analysis.
    • Developed and supported an MS Excel tool to provide Reliability Growth Analysis (RGA) using the Duane Model.
    • Facilitated various DFMEA sessions with program managers and design engineers, which fed into test plans and drove Design for Reliability.
    • Worked with test engineers to develop and monitor testing for a new compressor design and developed the chi2 zero-failure Reliability Metric used to communicate the programs progress to upper management.
